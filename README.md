# About
App for tracking and managing personal finances.

Heavily inspired by [AbilityCash](http://dervish.ru),
but has (or will have) some differences:
 - made on web technologies, so it will work on all platforms and devices
 - mobile-friendly
 - modern ui
 - cloud-based data storage
 - multiple users

## Install
    yarn install

## Dev
    yarn dev
    
## Build
    yarn build
